Queries

characters

Returns [Character!]!

Example

Query
query characters {
  characters {
    _id
    name
  }
}
Response 
{"data": {"characters": [{"_id": "xyz789", "name": "abc123"}]}}

episodes

Returns [Episode!]!

Example

Query
query episodes {
  episodes {
    _id
    name
    synopsis
    series {
      ...SeriesFragment
    }
    date
  }
}
Response 
{
  "data": {
    "episodes": [
      {
        "_id": "xyz789",
        "name": "abc123",
        "synopsis": "xyz789",
        "series": Series,
        "date": DateTime
      }
    ]
  }
}

quotes

Returns [Quote!]!

Example

Query
query quotes {
  quotes {
    _id
    episode {
      ...EpisodeFragment
    }
    characters {
      ...CharacterFragment
    }
    lines
  }
}
Response 
{
  "data": {
    "quotes": [
      {
        "_id": "abc123",
        "episode": Episode,
        "characters": [Character],
        "lines": ["xyz789"]
      }
    ]
  }
}

randomQuote

Returns a Quote!

Example

Query
query randomQuote {
  randomQuote {
    _id
    episode {
      ...EpisodeFragment
    }
    characters {
      ...CharacterFragment
    }
    lines
  }
}
Response 
{
  "data": {
    "randomQuote": {
      "_id": "xyz789",
      "episode": Episode,
      "characters": [Character],
      "lines": ["abc123"]
    }
  }
}

series

Returns [Series!]!

Example

Query
query series {
  series {
    _id
    name
    premiere
    finale
  }
}
Response 
{
  "data": {
    "series": [
      {
        "_id": "xyz789",
        "name": "xyz789",
        "premiere": DateTime,
        "finale": DateTime
      }
    ]
  }
}

String

The String scalar type represents textual data, represented as UTF-8 character sequences. The String type is most often used by GraphQL to represent free-form human-readable text.
