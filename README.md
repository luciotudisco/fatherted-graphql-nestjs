## Father Ted GraphQL API

A GraphQL server that provides API to retrieve hand-curated Father Ted quotes built using [Nest](https://github.com/nestjs/nest) and [MongoDB](https://www.mongodb.com/).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
